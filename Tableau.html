Tableau Web Dashboard of International Projects

This is a Tableau Web dashboard I made while working as a research assistant for the University of Leeds. I cannot change the format for my webpage as I have formatted it for the Universities page where it is also currently displayed. The beauty in this web dashboard is that it is linked with a google sheet, so my manager who is unfamiliar with tableau can still update and make changes to the map just by adding or removing entries to the excel sheet.

The majority of my work as a research assistant revolved around webscapping data and completeing queries, either through sql or pandas. Then I would either write reports to be sent to the board of pro-deans or create visualisations. In this case it was the first time I got to use tableau to create a dashboard for the internet.

As I am quite familiar with Tableau the dashboard was fairly straight forward to make, and as it turns out, it's fairly straightforward to publish to a website as well! thanks to Tableau public.
